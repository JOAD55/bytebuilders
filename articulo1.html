Historia del software


El software es el conjunto de programas, instrucciones y datos que permiten que una computadora o dispositivo realice tareas específicas. A diferencia del hardware (componentes físicos), el software es intangible y actúa como intermediario entre el usuario y la máquina.

Antecedentes del software


El desarrollo del software está ligado a la evolución de la computación. Aquí algunos hitos clave:

1. Siglo XIX - Primeros algoritmos:
- Ada Lovelace (1843) escribió el primer algoritmo para la máquina analítica de Charles Babbage, considerándose la primera programadora.

2. Década de 1940 - Origen del software moderno:
- Con las primeras computadoras electrónicas (como ENIAC), el software se manejaba mediante cableado físico.
- Se usaban lenguajes de bajo nivel (código máquina).

3. Década de 1950 - Lenguajes de programación:
- Aparece el ensamblador y luego lenguajes como FORTRAN (1957) y COBOL (1959), facilitando la programación.
- Nace el término "software" (acuñado por John Tukey en 1958).

4. Década de 1960 - Sistemas operativos y crisis del software:
- Surgen sistemas operativos primitivos (como UNIX).
- La complejidad del software llevó a la "crisis del software" por fallos en desarrollo y mantenimiento.

5. Década de 1970-1980 - Software comercial y personal:
- Empresas como Microsoft (fundada en 1975) impulsan el software para PCs.
- Nacen sistemas operativos como MS-DOS y lenguajes como C.

6. Década de 1990 - Expansión e internet:
- Aparece el software libre (Linux, 1991) y el modelo open source.
- La web impulsa aplicaciones como navegadores (Netscape) y herramientas empresariales.

7. Siglo XXI - Cloud, IA y móviles:
- Dominio del software como servicio (SaaS).
- Avances en inteligencia artificial y apps móviles.

Tipos de software


- De sistema: Sistemas operativos (Windows, Linux).
- De aplicación: Programas para usuarios (Word, Photoshop).
- De desarrollo: Herramientas para crear software (compiladores, IDEs).
- Embebido: Controla dispositivos específicos (ej. en electrodomésticos).

El software sigue evolucionando con tecnologías como computación cuántica, blockchain y metaverso.
